Evolution Game


The game (also called 'Conway's Game of Life') is a two-dimensional orthogonal grid of square cells, each can have two possible states, live or dead. Every cell interacts with its eight neighbours, horizontally, vertically, or diagonally adjacent.

Every generation evolves according to the following 4 rules:
1. Any live cell with fewer than two live neighbours dies, as if by underpopulation.
2. Any live cell with two or three live neighbours lives on to the next generation.
3. Any live cell with more than three live neighbours dies, as if by overpopulation.
4. Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.

Got the rules? Play now!

Set the initial configuration of cells by clicking few adjacent cells and click the start button. Or click Random button and click start.
The initial configuration you set determines the pattern that evolves later (Chaos Theory). Small changes in the initial conditions lead to drastic changes in the results (Butterfly effect). Share your patters with others. Refresh the page for a fresh start. Use desktop for better experience.
